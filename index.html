Frontend Web Dev

Frontend Web development has three major components to it:
  • HTML
    •
  • CSS
    •
  • JavaScript
    •
First, lets talk about the HTML. This is what gives structure to any sort of webpage. It makes the foundation or the bones that the whole page can stand on. It helps separate sections in a webpage that separates the various contents from one another.
Then there is CSS. This is what gives decoration to the structure of HTML. It is the skin to the bones of HTML.
Finally, we have JavaScript. It finally gives life to the whole page by turning it from a static webpage to a interactive web application.
This was an explanation of Frontend Web Development! You can go to the Coursera course to learn a lot more about these things.
